---
### 👨🏼‍💻Функціонал:

**Hardware PWM Generation:** Генерація ШІМ-сигналу на піні PA6 (TIM3, Channel 1) з фіксованою частотою 500 Гц. Використання апаратного таймера дозволяє підтримувати стабільний сигнал без залучення ресурсів центрального процесора.

**Interrupt-Driven Control (EXTI):** Обробка натискання користувацької кнопки на PA0 через зовнішнє переривання. Це забезпечує миттєву реакцію системи без використання затримок HAL_Delay у головному циклі.

**Dynamic Duty Cycle:** Ступінчасте регулювання яскравості світлодіода (або потужності навантаження) у діапазоні від 10% до 90% з кроком 20%.

**Formula for PWM Frequency:** 
При тактуванні таймерів $F_{TIM} = 100$ МГц, параметри розраховані як:

$$F_{PWM} = \frac{100,000,000}{(99 + 1) \cdot (1999 + 1)} = 500 \text{ Гц}$$

---
### 🔵Підключення:

**1. Кнопка (Вхід керування):**

- PA0 — вбудована кнопка "KEY" на Black Pill (замикає на GND при натисканні). Налаштована з внутрішньою підтяжкою Pull-up.

**2. Світлодіод (Вихід ШІМ):**

- Cathode (-) — до GND.

- Anode (+) — через резистор (~325 Ом) до піна PA6.

---
### 🛠️ Налаштування периферії

Для цієї роботи реалізовано апаратну генерацію ШІМ у поєднанні з перериваннями EXTI.

**Чому це ефективно:**

- **Автономність:** Таймер 3 самостійно керує станом піна PA6. Навіть якщо програма "зависне", ШІМ продовжуватиме працювати.

- **Відсутність блокування:** Замість постійного опитування кнопки в while(1), процесор "спить" або виконує інші задачі. Він прокидається лише в момент натискання (Falling Edge), виконує коротку функцію Callback і миттєво повертається до основної роботи.

- **Програмний антидребезг:** Реалізовано через перевірку системних тіків HAL_GetTick(), що дозволяє ігнорувати помилкові спрацьовування контактів без зупинки всього мікроконтролера.

### ⁉️Як запустити:
1. Відкрийте проект у STM32CubeIDE/Keil.
2. Переконайтеся, що в Clock Configuration встановлено HSE 25 MHz та HCLK 100 MHz.
3. Скомпілюйте проект (Build) та прошийте плату через ST-Link.
4. Натискайте кнопку на PA0 для циклічної зміни яскравості світлодіода.
